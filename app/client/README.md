# UCM Client App

Status: product boundary and build plan established; authenticated application not yet implemented.

The Client App will give approved UCM customers secure access to CleanScope enquiries, human-reviewed quote status, bookings, Proof Passports, documents and support. It remains separate from the public website and internal CRM.
