# UCM Media Engine Adaptation

The workspace already contains a useful media-ops pattern in the Unroast project. UCM should adapt the structure, not the brand content.

## Existing Pattern Worth Reusing

The existing media ops engine assumes:

- Source watching.
- Ingest.
- Whisper transcription.
- Segmenting.
- 15-60 second clip generation.
- Approval queue.
- YouTube publishing.
- TikTok scaffold.
- Metrics.
- Learning loop.

## UCM Version

UCM needs a service-business media center:

1. Watch source folders for job photos, site videos, founder voice notes and customer proof.
2. Tag every asset by borough, service, buyer segment and proof status.
3. Generate campaign-ready content packets.
4. Create short-form video briefs and captions.
5. Check claims before publishing.
6. Route assets through approval.
7. Return metrics by borough, service, hook, CTA and lead quality.

## UCM Content Packet

Every content packet should include:

- campaign_id
- borough_or_zone
- service
- buyer_segment
- source_asset_path
- proof_status
- hook
- script
- caption
- CTA
- landing_page
- platform_targets
- approval_owner
- reviewer
- publish_status

## Approval Rules

No asset publishes unless:

- The service claim is accurate.
- Any AI-savings claim uses approved wording.
- Before/after images are real and approved.
- Customer review/testimonial is real and approved.
- Location targeting is accurate.
- CTA and landing page are set.

## Learning Loop

Return metrics by:

- zone
- borough
- service
- campaign_id
- platform
- hook
- CTA
- creative format
- views
- saves
- shares
- comments
- profile visits
- landing page clicks
- calls
- quote forms
- qualified leads

## Recommended Agent Roles

- Orchestrator: plans campaigns and assigns tasks.
- Researcher: updates London targeting and competitor signals.
- Media Worker: clips videos, drafts captions, prepares thumbnails.
- SEO/GEO Worker: turns campaign learning into landing pages and AI-search copy.
- Paid Media Worker: builds Meta campaign briefs and stop-loss checks.
- Advisor: reviews claims, economics and strategic focus before publishing.
