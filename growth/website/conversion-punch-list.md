# UCM Website Conversion Punch List

Priority: convert the current website from a polished brochure into a lead-generation asset.

## Critical Fixes

### 1. Create Crawlable Service Pages

Priority pages:

- Office Cleaning London.
- Commercial Cleaning London.
- Property Management Cleaning London.
- End of Tenancy Cleaning London.
- Data Centre Cleaning London.
- Responsive Property Maintenance London.

Each page needs unique metadata, indexable content, FAQs and clear quote CTA.

### 2. Add Sitemap And Structured Data

Recommended:

- XML sitemap.
- `LocalBusiness` schema.
- `Organization` schema.
- `Service` schema for priority services.
- `FAQPage` schema on service pages.
- `BreadcrumbList` schema.

### 3. Improve Above-The-Fold Proof

Homepage and service pages should show:

- Service clarity.
- Location relevance.
- Phone CTA.
- Quote CTA.
- Reviews/proof.
- Accreditations or compliance evidence.
- Buyer-specific promise.

### 4. Fix Mobile Hero Clarity

Observed issue:

- Mobile hero feels visually cramped and heavy.

Recommended:

- Reduce headline density.
- Add clearer CTA.
- Add visible phone/quote action.
- Make proof visible earlier.

### 5. Service-Specific Quote Forms

Quote form should capture:

- Service needed.
- Property type.
- Postcode.
- Urgency.
- Company.
- Phone.
- Email.
- Message.
- Permission to contact.

Track submissions by source page.

### 6. Add Trust Blocks

Needed:

- Real reviews.
- Case studies.
- Before/after photos.
- Accreditations.
- Insurance/compliance proof.
- Team/process photos.
- Service checklist.

### 7. Tracking

Track:

- Form submissions.
- Phone clicks.
- Quote button clicks.
- Service page views.
- Paid campaign landing page visits.
- Scroll depth on priority pages.

## First Implementation Recommendation

Build the `Office Cleaning London` page first because it is a high-intent, clear buyer journey and can support SEO, paid ads and social CTAs.
